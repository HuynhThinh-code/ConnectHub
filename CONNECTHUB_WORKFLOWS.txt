2. DATABASE WORKFLOW
--------------------

Bang chinh:

    - Luu tai khoan, role admin/user, avatar, gender, ban status, OAuth info.

    - Luu bai viet, anh, private flag, status moderation.

    - Luu tin nhan rieng giua users.

friend_requests
    - Luu loi moi ket ban va trang thai accepted/pending/rejected.

    - Log login, payload tan cong, AI Fix, block/unblock, reset lab.

    - Rule active de AI Fix bao ve route.
    - Bi reset khi bam Reset Lab State de lab quay lai vulnerable state.

    - Tri nho dai han cua ConnectHub Sentinel ve cach fix.
    - Khong phai rule active.

ai_attack_memory
    - Tri nho fingerprint attack la.
    - Lan dau: unknown_attack.
    - Sau khi admin bam Learn Attack: ghi memory.
    - Lan sau payload giong fingerprint: learned_attack.

ai_agent_config
    - Luu OpenAI API key va model cua ConnectHub Sentinel.

blocked_ips
    - IP bi admin block.
